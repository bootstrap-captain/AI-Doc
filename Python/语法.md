# 基本介绍

## 缺点

- 运行速度慢：解释型语言，代码运行时要通过解释器翻译成机器码再执行
- 代码无法加密： .py文件很容易被逆向查看代码逻辑，保护代码知识产权比较难
- 内存消耗大：更加吃内存

## AI领域

- 试错，调模型，调参数：使用Python可以更快速的实现，调试和维护比较简单
- Python的轮子较多：PyTorch, TensorFlow, Scikit-learn都是顶级的AI框架
- 与底层语言高校协助：AI框架表面是Python, 底层使用C或者C++实现

## 数据类型

```python

print(type(123))

# 数字可以用_区分

b = 1000
print(a == b)

firstName = "erick"
secondName = 'shu'
name = f'{firstName} {secondName}'

```

```python
# 给一个变量进行初始化，但是不对其做类型暗示
msg = None

# NoneType
print(type(msg))

# 转换为boolean是False
if msg:
    print('hello')
```

### list

- 数组：容器中的数据类型不做限制

```python
# 正向索引：0～2，   负索引：-3，-2，-1
fruits = ["apple", "banana", "cherry"]

# 新增:
   # 在尾部追加元素
fruits.append("mango")
   # 在指定索引处追加元素
fruits.insert(1, "grape")
   # 将可迭代对象中的内容追加到列表尾部: 字符串，分别为 g r a p e
fruits.extend("grape")
fruits.extend(["cc","dd"])

# 删除
result = fruits.pop(1)
fruits.clear()

print(fruits)
```

## 流程

```python
age = input("What is your age?")

if int(age) <= 10:
    print("You are young.")
elif 10 <= int(age) <= 20:
    print("You are a teenager.")
elif 20 <= int(age) <= 30:
    print("you are adult")
else:
    print("You are old")
```

```python
age = 0
while age <= 5:
    print(f"hello{age}")
    age += 1
```

```python
# 0~4: 左开右闭
for age in range(5):
    print(age)

print("==========")
# 2~4: 左开右闭
for num in range(2,5):
    print(num)
```

## 函数

```python
# 函数定义
  # 带默认值的参数可以不传递值，一般放在最后
def generateInfo(name, gender, address='nanjing'):
    print(f'name==={name}, gender==={gender}, address=={address}')


# 关键字参数调用
generateInfo(name="erick", address="xian", gender="male")

# 位置参数调用
generateInfo("lucy", "girl", "beijing")
```

## 文件

- try with resources 

```python
with open(file='a.txt', mode='r', encoding='utf-8') as config:
    result = config.read()
    print(result)
```
